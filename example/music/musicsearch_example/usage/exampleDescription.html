豆瓣 - 豆瓣音乐搜索

模板介绍
  • 此模板用于采集豆瓣音乐类目下的关键词搜索结果的信息，如专辑名称、演奏者、介绍、曲目等。
  • 具体采集内容请点击「采集字段预览」和「示例数据」。
使用方法
  1. 点击「立即使用」进入参数配置界面
  2. 数据的爬取页数（1~99）（e.g.输入3表示爬取搜索结果1-3页的内容）
  3. 输入需要搜索的关键字
    注：
    • 如要输入多个关键字，请在每个关键字之间用换行分割
    • 可从Excel表格和TXT文件中直接复制影名，黏贴到输入框中
    • 建议一次性输入的关键词不要超过10个
  4. 参数设置完毕后，请点击「保存并启动」进行采集
